- Validasi saat registrasi (cek username sudah digunakan atau belum).

- Validasi login (username & password harus cocok dengan data di database).

- Navigasi antar form, seperti kembali dari form registrasi ke login.

- Struktur kode yang mengikuti konsep OOP dan arsitektur yang bersih.
